# build your own ai

## uv 查找 Python 环境的顺序

1. 目前目录下的 `.python-version` 文件设定的版本；
2. 目前启用的虚拟环境；
3. 目前目录下的 `.venv` 目录内设定的虚拟环境；
4. uv 自己安装的 python 环境；
5. 系统环境设定的 python 环境；
