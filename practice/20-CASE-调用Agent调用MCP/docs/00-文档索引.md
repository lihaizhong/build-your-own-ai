## 🔍 快速查找

### 📌 **常见问题解答**

#### **Q: 我应该从哪个文档开始？**
**A**: 新手请从 [项目概述](01-项目概述.md) 开始，开发者请从 [技术架构](02-技术架构.md) 开始。

#### **Q: 如何快速上手使用？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分，运行：
```bash
python code/main.py
```

#### **Q: 如何开发自定义 MCP 服务？**
**A**: 参考 [代码实现](04-代码实现.md) 中的自定义 MCP Server 部分。

#### **Q: 支持哪些 MCP 服务？**
**A**: 目前支持高德地图、Fetch、Bing 搜索、自定义 TXT 统计等，详见 [技术架构](02-技术架构.md#mcp-服务集成)。

#### **Q: 如何切换到终端模式？**
**A**: 修改 `main.py` 中的 `__main__` 部分调用 `app_tui()`，详见 [使用指南](03-使用指南.md#终端交互模式)。
