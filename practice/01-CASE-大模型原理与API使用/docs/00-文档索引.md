### 📌 **常见问题解答**

#### **Q: 我应该从哪个文档开始？**
**A**: 新手请从 [项目概述](01-项目概述.md) 开始，有经验的开发者请从 [技术架构](02-技术架构.md) 开始。

#### **Q: 如何快速运行第一个示例？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分，运行 `uv run python code/1-情感分析-Qwen.py`。

#### **Q: Function Calling是什么？**
**A**: 详见 [技术架构](02-技术架构.md#function-calling)，Function Calling允许大模型调用外部工具获取实时数据。

#### **Q: 如何配置API密钥？**
**A**: 参考 [使用指南](03-使用指南.md#环境准备)，需要在 `.env` 文件中配置 `DASHSCOPE_API_KEY`。

#### **Q: 支持哪些模型？**
**A**: 项目支持多种模型，详见 [项目概述](01-项目概述.md#支持模型)。
