# 二手车价格预测项目 - 项目概述

## 项目简介

二手车价格预测项目是一个完整的机器学习应用系统，专注于使用集成学习方法预测二手车的交易价格。该项目基于阿里云天池竞赛数据，通过多轮迭代优化，从初始MAE 1220+逐步改进到更稳健的预测模型。

项目包含了完整的数据预处理、特征工程、模型训练和性能评估流程，展示了机器学习在二手车交易场景中的实际应用能力。

## 核心功能

### 1. 数据预处理
- 完整的数据清洗管道
- 缺失值智能处理
- 异常值识别与处理
- 分类特征编码策略

### 2. 特征工程
- 时间特征提取（车龄、注册月份等）
- 业务逻辑特征（功率/车龄比、年公里数等）
- 统计特征生成（品牌均价、价格稳定性等）
- 交互特征构建

### 3. 模型训练
- LightGBM模型训练
- XGBoost模型训练
- CatBoost模型训练
- 模型集成与融合策略

### 4. 预测与评估
- 多模型加权集成预测
- 分位数校准优化
- MAE/RMSE评估指标
- 分布校准验证
