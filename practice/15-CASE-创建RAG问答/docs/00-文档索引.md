## 🔍 快速查找

### 📌 **常见问题解答**

#### **Q: 我应该从哪个文档开始？**
**A**: 新手请从 [项目概述](01-项目概述.md) 开始，开发者请从 [技术架构](02-技术架构.md) 开始。

#### **Q: 如何快速上手使用？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分，运行 `uv run python code/main.py`。

#### **Q: 如何配置API密钥？**
**A**: 在项目根目录创建 `.env` 文件，配置 `DASHSCOPE_API_KEY`。详见 [使用指南](03-使用指南.md#环境准备)。

#### **Q: 如何更换PDF文档？**
**A**: 将新的PDF文件放入 `data/` 目录，通过命令行参数指定路径。详见 [使用指南](03-使用指南.md#自定义文档处理)。

#### **Q: 如何部署到生产环境？**
**A**: 参考 [测试和部署](05-测试和部署.md) 中的部署策略部分。
