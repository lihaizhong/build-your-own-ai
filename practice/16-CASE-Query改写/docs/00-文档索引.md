## 🔍 快速查找

### 📌 **常见问题解答**

#### **Q: Query改写是什么？**
**A**: Query改写是将用户的原始查询转换为更适合检索系统处理的查询形式，提升RAG系统的检索效果。

#### **Q: 支持5种Query类型分别是什么？**
**A**: 上下文依赖型、对比型、模糊指代型、多意图型、反问型。详见 [项目概述](01-项目概述.md#query类型分类)。

#### **Q: 如何快速上手？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分，运行 `python code/main.py`。

#### **Q: 如何与现有RAG系统集成？**
**A**: 参考 [使用指南](03-使用指南.md#与rag系统集成) 中的集成方案和代码示例。

#### **Q: 改写效果不好怎么办？**
**A**: 参考 [使用指南](03-使用指南.md#最佳实践) 中的调优建议和Prompt优化技巧。
