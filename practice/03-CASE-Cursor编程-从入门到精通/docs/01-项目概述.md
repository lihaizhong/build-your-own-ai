# Cursor编程从入门到精通 - 项目概述

## 项目简介

Cursor编程从入门到精通项目是一个完整的数据处理与可视化实践项目，演示了如何使用AI编程工具（Cursor、Trae、CodeBuddy等）快速开发数据处理和可视化应用。该项目涵盖Excel数据读取、数据合并处理、以及Flask可视化大屏等多个实战案例。

## 核心功能

### 1. Excel数据处理
- 员工信息表读取与展示
- 多表数据合并与关联
- 数据验证与统计分析
- 自动生成示例数据

### 2. 数据合并处理
- 员工基本信息与绩效数据合并
- 基于员工编号的左连接操作
- 数据统计摘要生成
- 部门绩效分析报表

### 3. 疫情数据可视化大屏
- 香港各区疫情数据展示
- ECharts地图可视化
- 多维度数据分析
- 实时数据更新

### 4. 医院病床可视化大屏
- 各科室病床使用情况
- 占用率趋势分析
- 可用床位实时监控
- 热力图数据展示

## 案例说明

### 案例1：员工数据合并
- **输入**: 员工基本信息表.xlsx + 员工绩效表.xlsx
- **处理**: 基于员工编号左连接合并
- **输出**: 员工综合信息表_2024Q4.xlsx
- **特点**: 自动生成统计摘要和绩效分析

### 案例2：疫情可视化大屏
- **数据源**: 香港各区疫情数据
- **展示**: 地图分布 + 趋势图表 + 排行榜
- **技术**: Flask + ECharts
- **特点**: 5个核心图表联动展示

### 案例3：医院病床大屏
- **数据源**: 医院科室病床数据
- **展示**: 占用率 + 趋势 + 可用床位
- **技术**: Flask + ECharts
- **特点**: 实时监控与预警
