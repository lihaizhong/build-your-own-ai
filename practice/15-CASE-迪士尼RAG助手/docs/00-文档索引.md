## 🔍 快速查找

### 📌 **常见问题解答**

#### **Q: 我应该从哪个文档开始？**
**A**: 新手请从 [项目概述](01-项目概述.md) 开始，开发者请从 [技术架构](02-技术架构.md) 开始。

#### **Q: 如何快速上手使用？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分：
```bash
python -m code.main --build       # 构建索引
python -m code.main --interactive # 交互问答
```

#### **Q: 如何添加自己的文档？**
**A**: 将文档放入 `data/documents/` 目录，图像放入 `data/images/` 目录，然后重新运行 `python -m code.main --build`。

#### **Q: 支持哪些文档和图像格式？**
**A**: 文档支持 `.docx`、`.txt`、`.md`；图像支持 `.png`、`.jpg`、`.jpeg`、`.gif`、`.bmp`、`.webp`。

#### **Q: 如何触发图像检索？**
**A**: 查询中包含关键词如"海报"、"图片"、"图像"、"照片"等即可触发图像检索。
