## 🔍 快速查找

### 📌 **常见问题解答**

#### **Q: 我应该从哪个文档开始？**
**A**: 新手请从 [项目概述](01-项目概述.md) 开始，开发者请从 [技术架构](02-技术架构.md) 开始。

#### **Q: 如何快速上手使用？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分，运行：
```bash
python code/init_db.py
python code/main.py
```

#### **Q: 如何添加自定义查询工具？**
**A**: 参考 [使用指南](03-使用指南.md) 中的自定义扩展部分，以及 [代码实现](04-代码实现.md) 中的工具实现详解。

#### **Q: 支持哪些数据库？**
**A**: 目前支持 SQLite 和 MySQL，详见 [技术架构](02-技术架构.md#数据库连接架构)。

#### **Q: 如何切换到终端模式？**
**A**: 运行 `python code/main.py --mode tui`，详见 [使用指南](03-使用指南.md#终端交互模式)。
