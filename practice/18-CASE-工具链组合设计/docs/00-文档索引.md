## 🔍 快速查找

### 📌 **常见问题解答**

#### **Q: 我应该从哪个文档开始？**
**A**: 新手请从 [项目概述](01-项目概述.md) 开始，开发者请从 [技术架构](02-技术架构.md) 开始。

#### **Q: 如何快速上手使用？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分，运行 `python code/main.py --mode demo`。

#### **Q: 如何配置 API 密钥？**
**A**: 在项目根目录创建 `.env` 文件，添加 `DASHSCOPE_API_KEY=your_api_key` 或 `OPENAI_API_KEY=your_api_key`。

#### **Q: 支持哪些 LLM 提供商？**
**A**: 支持通义千问(Tongyi)、OpenAI 两种提供商。

#### **Q: 如何添加自定义工具？**
**A**: 参考 [代码实现](04-代码实现.md) 中的工具类实现，然后在 Agent 中注册。
