# 波士顿房价预测项目 - 项目概述

## 项目简介

波士顿房价预测项目是一个完整的机器学习应用系统，演示了如何使用多种机器学习算法进行房价预测。该项目基于经典的波士顿房价数据集，涵盖了从数据预处理、特征工程、模型训练到模型评估和预测的完整机器学习流程。

## 核心功能

### 1. 数据预处理
- 数据加载和质量检查
- 缺失值处理
- 异常值检测
- 特征统计分析

### 2. 特征工程
- 多项式特征创建
- 特征相关性分析
- 特征选择和重要性分析
- 特征标准化（Standard、Robust、MinMax）
- 交互特征创建

### 3. 模型训练
- 多种机器学习算法支持
  - 线性模型：LinearRegression、Ridge、Lasso
  - 集成学习：RandomForest、GradientBoosting
  - 支持向量机：SVR
  - 神经网络：MLPRegressor
- 超参数调优
- 交叉验证
- 最佳模型选择

### 4. 模型评估
- 多维度评估指标
  - MAE（平均绝对误差）
  - MSE（均方误差）
  - RMSE（均方根误差）
  - R²（决定系数）
  - Explained Variance（解释方差）
- 可视化分析
  - 预测值vs实际值图
  - 残差分析图
  - 模型对比图

### 5. 房价预测
- 单样本预测
- 批量预测
- 模型预测对比
- 预测结果保存

## 数据集说明

### 波士顿房价数据集特征

| 特征名 | 描述 | 类型 |
|--------|------|------|
| CRIM | 城镇人均犯罪率 | 连续型 |
| ZN | 住宅用地超过25,000平方英尺的比例 | 连续型 |
| INDUS | 城镇非零售商业用地比例 | 连续型 |
| CHAS | 是否靠近查尔斯河（1=是，0=否） | 二元型 |
| NOX | 一氧化氮浓度（百万分之一） | 连续型 |
| RM | 每栋住宅平均房间数 | 连续型 |
| AGE | 1940年前建造的自住单元比例 | 连续型 |
| DIS | 到波士顿五个就业中心的加权距离 | 连续型 |
| RAD | 到径向高速公路的可达性指数 | 连续型 |
| TAX | 每10,000美元的全额财产税率 | 连续型 |
| PTRATIO | 城镇师生比例 | 连续型 |
| B | 1000(Bk - 0.63)²，Bk为黑人比例 | 连续型 |
| LSTAT | 低收入人口比例 | 连续型 |
| MEDV | 自住房屋中位价格（目标变量） | 连续型 |

### 数据集统计
- **样本数量**: 506条
- **特征数量**: 13个
- **目标变量**: 房屋中位价格（千美元）
