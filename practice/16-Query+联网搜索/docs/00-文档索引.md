## 🔍 快速查找

### 📌 **常见问题解答**

#### **Q: 我应该从哪个文档开始？**
**A**: 新手请从 [项目概述](01-项目概述.md) 开始，开发者请从 [技术架构](02-技术架构.md) 开始。

#### **Q: 如何快速上手使用？**
**A**: 参考 [使用指南](03-使用指南.md) 中的快速开始部分，运行 `uv run python code/main.py`。

#### **Q: 如何配置API密钥？**
**A**: 在项目根目录创建 `.env` 文件，添加 `DASHSCOPE_API_KEY=your_api_key`。

#### **Q: 如何集成到现有的RAG系统？**
**A**: 参考 [使用指南](03-使用指南.md) 中的RAG集成示例代码。

#### **Q: 支持哪些搜索类型？**
**A**: 支持6种搜索类型：时效性、天气、新闻资讯、价格行情、实时数据、不需要联网。
