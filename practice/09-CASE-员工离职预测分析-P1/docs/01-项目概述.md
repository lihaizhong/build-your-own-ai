# 员工离职预测分析项目 - 项目概述

## 项目简介

员工离职预测分析项目是一个完整的人力资源分析系统，利用机器学习技术预测员工离职风险。该项目基于员工的历史数据，通过数据探索、特征工程和模型训练，帮助企业提前识别高风险员工，从而采取针对性的留人措施。

## 核心功能

### 1. 数据探索性分析（EDA）
- 全面的数据质量检查
- 目标变量分布分析
- 数值特征统计和相关性分析
- 分类特征分布可视化
- 特征与目标变量关系探索

### 2. 特征工程
- 多维度特征构建
- 分类变量编码
- 特征重要性分析
- 特征选择和降维

### 3. 离职预测模型
- 多种算法对比实验
- 交叉验证和模型评估
- 超参数调优
- 模型解释性分析

### 4. 预测结果输出
- 测试集预测
- 预测结果可视化
- 风险等级划分
- 预测报告生成

## 数据集介绍

### 数据概述
- **训练集**: 1177条员工记录
- **测试集**: 待预测员工记录
- **特征数量**: 35个特征
- **目标变量**: Attrition（是否离职：Yes/No）

## 技术栈

### 核心技术
- **Python 3.11+**: 主要编程语言
- **Pandas**: 数据处理和分析
- **NumPy**: 数值计算
- **Scikit-learn**: 机器学习算法
- **Matplotlib**: 数据可视化
- **Seaborn**: 统计数据可视化

### 数据处理
- **数据清洗**: 缺失值处理、异常值检测
- **特征编码**: LabelEncoder、OneHotEncoder
- **特征缩放**: StandardScaler、MinMaxScaler
- **特征选择**: 基于重要性、相关性筛选

### 机器学习算法
- **逻辑回归**: 基线模型
- **决策树**: 可解释性模型
- **随机森林**: 集成学习
- **XGBoost/LightGBM**: 梯度提升
- **CatBoost**: 类别特征优化

### 模型评估
- **准确率（Accuracy）**: 整体预测准确度
- **精确率（Precision）**: 预测为正的准确度
- **召回率（Recall）**: 正样本召回能力
- **F1分数**: 精确率和召回率的调和平均
- **ROC-AUC**: 综合评估指标

## 性能指标

### 数据质量
- **缺失值**: 无缺失值
- **数据类型**: 混合类型（数值+分类）
- **特征数量**: 35个特征
- **样本数量**: 1177条训练数据

### 目标变量分布
- **离职（Yes）**: 约16%的样本
- **未离职（No）**: 约84%的样本
- **类别不平衡**: 需要采用相应策略处理

### 模型性能目标
- **准确率目标**: >85%
- **召回率目标**: >70%（关注离职预测）
- **AUC目标**: >0.80
- **F1分数目标**: >0.75
