# 三国演义词嵌入分析项目 - 项目概述

## 项目简介

三国演义词嵌入分析项目是一个专注于中文古典文学作品《三国演义》的词向量研究和应用系统。通过Word2Vec技术将文本中的词汇转换为高维向量表示，实现词汇语义关系的挖掘、人物关系分析、词类比推理等功能。

本项目不仅展示了词嵌入技术的实际应用，也为中文NLP研究提供了有价值的参考案例。

## 核心功能

### 1. 词向量训练
- 使用Gensim Word2Vec训练词向量模型
- 支持CBOW和Skip-gram两种训练模式
- 可配置的向量维度和训练参数
- 支持增量训练和模型保存

### 2. 词相似度分析
- 计算词汇之间的语义相似度
- 发现语义相近的词汇
- 支持人物关系挖掘
- 可视化相似度矩阵

### 3. 词类比计算
- 基于向量运算的类比推理
- 支持"A:B = C:?"类型的推理
- 发现隐藏的语义关系
- 支持多词汇组合运算

### 4. 词汇关系可视化
- 词向量降维可视化
- 相似词聚类展示
- 人物关系网络图
- 语义空间探索

## 分析成果

### 1. 人物关系发现
通过词向量分析，成功发现三国人物之间的语义关联：
- 曹操与马超、袁术、周瑜等对手关联度高
- 刘备与诸葛亮、关羽、张飞等关联紧密
- 孙权与东吴将领关联明显

### 2. 语义类比验证
词类比计算结果符合历史认知：
- "曹操 + 刘备 - 张飞 ≈ 明公/孤"（曹操的常用自称）
- "诸葛亮 + 刘备 - 关羽" 得到有意义的关联词
- 模型成功捕捉到人物特征和关系

### 3. 词频分布分析
- 高频词以标点和文言虚词为主
- "曰"（说）是高频动词，反映对话体特征
- 人物名出现频率与戏份正相关

## 快速开始

### 环境准备
```bash
# 进入项目目录
cd practice/14-CASE-三国演义Embedding

# 创建虚拟环境
uv venv --python 3.11

# 激活虚拟环境
source .venv/bin/activate

# 安装依赖
uv sync
```

### 运行训练
```bash
# 训练Word2Vec模型
uv run python code/three_kingdoms.py

# 运行演示程序
uv run python code/word2vec_demo.py
```

### 使用模型
```python
from gensim.models import Word2Vec

# 加载模型
model = Word2Vec.load('model/three_kingdoms.model')

# 查找相似词
similar_words = model.wv.most_similar('曹操', topn=10)

# 词类比计算
result = model.wv.most_similar(
    positive=['曹操', '刘备'],
    negative=['张飞'],
    topn=5
)
```
